## Reporting Issues

If you discover a bug or security vulnerability:

1. Open an issue using the appropriate template
2. Provide clear reproduction steps and relevant details
3. Include error messages or logs if applicable
4. **Do not create pull requests to fix issues**

We will review and address reported issues internally.
